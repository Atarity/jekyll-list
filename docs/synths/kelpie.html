Portable polyphonic digital synthesizer.

The device consists of 16 knobs and 4 LED buttons, a standard MIDI input, 1/4 mono output, 1/8 stereo output, and a micro-USB input (for programming). The goal of this project was to create a budget digital synthesizer for all experience levels that could be easily interfaced with a MIDI keyboard and used in live performances. This is a no-frills synthesizer, with no presets, arpeggiator, or storage.
