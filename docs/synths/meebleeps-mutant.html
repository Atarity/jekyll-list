8-Bit 2-oscillator subtractive Arduino synth for generative techno, using Mozzi library, in Volca form-factor.

Synth Voice Features - 2 saw oscillators - Multiple tuning modes for 2nd oscillator - off, detune, fifths & octaves up/down - Digital low pass filter with variable cutoff and resonance - Variable level ducking/sidechain effect Sequencer - Generative sequencer mutates/evolves at user-defined rate - Variable sequence mutation probability & note-density - Variable sequencer length (1-16 steps) - Sync input & output (Korg Volca compatible) - Selectable tonic note - Selectable scale quantisation (Major, Minor, Pentatonic, Phrygian (GOA!), Octaves, Fifths) - 16-step parameter-lock recording of synth parameters (the Elektron way!) - Retrig (clone) button for fills - Tap-tempo control More details in Reddit post .
